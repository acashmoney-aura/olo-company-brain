# Olo Company Brain

Outward-looking company brain for Olo built from public information and structured as a KRAIL company-brain project.

## Scope

This repo models Olo as an external analyst would understand it from public sources:
- company identity and timeline
- product map
- customer and buyer map
- business model and external dependencies
- competitor landscape
- claims with evidence, confidence, and open questions

## Why Olo

Olo is a good proof-of-concept company because it is small enough to map end-to-end, but public enough to have substantial external evidence across product pages, investor materials, acquisition announcements, partner surfaces, and industry coverage.

## Current focus

- seed an external Olo knowledge base from high-trust public sources
- test KRAIL company-brain workflows on a real company repo
- document package failures and rough edges as reproducible GitHub issues
